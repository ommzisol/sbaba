# MarketPulse Pro

This repository now focuses on a single Next.js project based on the `template-01` design.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind utility classes loaded through the CDN in the root layout

## Main Routes

- `/`
- `/about`
- `/contact`
- `/support`
- `/privacy-policy`
- `/terms`
- `/refund-policy`
- `/cookies-policy`

## Run Locally

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Notes

- The old `templates/` directory is still present as reference material for now, but the active app is the new Next.js site in `app/`, `components/`, and `lib/`.
- The previous multi-template launcher logic is no longer part of the main application flow.
